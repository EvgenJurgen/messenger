# Feature-Sliced Design (FSD)

Слои (снизу вверх, импорт только вниз):

```
app          — инициализация, провайдеры, роутинг, глобальные стили
  ↓
pages        — страницы (композиция виджетов и фич)
  ↓
widgets      — составные блоки (header с языком и темой)
  ↓
features     — сценарии (auth: логин, регистрация, сессия)
  ↓
entities     — бизнес-сущности (user)
  ↓
shared       — ui, api, lib (переиспользуемый код)
```

## Алиас

- `@/` → `src/` (например: `@/shared/ui`, `@/features/auth`)

## Публичный API

Каждый слайс экспортирует только через `index.ts` (баррель).
